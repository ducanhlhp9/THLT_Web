Phần 1: tạo Cơ sở dữ liệu và import dữ liệu
1. Tạo cơ sở dữ liệu THLTWEB
2. chạy "php artisan migrate" trong terminal của BackEnd để tạo các bảng trong cơ sở dữ liệu
3. Import các file ".sql" vào để tạo dữ liệu

Phần 2: Khởi tạo server

Chạy câu lệnh :"php artisan serve" trong terminal của Backend để chạy Server 
Server sẽ chạy trên local host cổng 8000: http://127.0.0.1:8000
(Tùy từng máy có thể sẽ phải cài thêm các môi trường khác nhau)

Phần 3: Khởi chạy giao diện trang chủ

chạy câu lệnh :"npm start" trong terminal của Front_End để chạy
Giao diện sẽ chạy trên localhost cổng 3000: http://localhost:3000

Phần 4: Khởi chạy giao diện Admin

Chạy câu lệnh :"npm start" trong terminal của FrontEnd_admin để chạy
Giao diện sẽ chạy trên localhost cổng 3001: http://localhost:3001
